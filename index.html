FileXShared

Share files Fast and Easy, on your local network!

Requires

Written for Python3

Install

PIP: sudo pip install FileXShared

Manual Installation: sudo python setup install

Run

  1. Run in Terminal: FileXShared
  2. Open localhost:8054
